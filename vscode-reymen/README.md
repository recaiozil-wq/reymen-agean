# ReYMeN VS Code Extension

ReYMeN AI Agent entegrasyonu.

## Kurulum

1. VS Code'da `Extensions` paneline gidin
2. `...` menu > `Install from VSIX...`
3. `vsce package` ile paketlenmis dosyayi secin

## Kullanim

- `Ctrl+Shift+R` / `Cmd+Shift+R` — Secili kodu ReYMeN'e sor
- `ReYMeN: Sohbet` komutu — Webview sohbet paneli
- `ReYMeN: Durum Kontrol` — Baglanti testi

## Gereksinimler

- Calisan bir ReYMeN MCP sunucusu (varsayilan: http://localhost:9000)
