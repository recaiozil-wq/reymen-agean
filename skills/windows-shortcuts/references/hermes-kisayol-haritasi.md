# ReYMeN Kısayol Haritası

Bu referans, ReYMeN Agent'in Windows otomasyon script'leri ve bunlara atanmış klavye kısayollarını içerir.

## Özel Scriptler — AutoHotkey ile `Ctrl+Alt+Harf`

| Kısayol | Script | Ne Yapar |
|---------|--------|----------|
| `Ctrl+Alt+M` | hermesmouse.py | Fare/klavye simülasyonu |
| `Ctrl+Alt+O` | approve_popup.py | Popup/dialog onaylama |
| `Ctrl+Alt+V` | capture_screen_temp.ps1 | Ekran görüntüsü al |
| `Ctrl+Alt+K` | SSH → Kali | Kali VM'e bağlan |
| `Ctrl+Alt+H` | SSH → Kali (apropos) | Kali araç yardımı |
| `Ctrl+Alt+W` | VBoxManage | Kali USB WiFi tarama |
| `Ctrl+Alt+T` | vm_web_terminal.py | Web terminal başlat |
| `Ctrl+Alt+Q` | ollama_start.py | Ollama başlat |
| `Ctrl+Alt+A` | code | VS Code aç |
| `Ctrl+Alt+B` | Tor Browser | Tor Browser aç |
| `Ctrl+Alt+I` | CLI installer | Python CLI aracı kur |
| `Ctrl+Alt+R` | state.db temizlik | Hafıza temizleme |
| `Ctrl+Alt+N` | notepad .env | .env dosyasını düzenle |
| `Ctrl+Alt+S` | sync_skills_to_obsidian.py | Skill + Obsidian senkronizasyonu |
