# Harmonic Mixing & DJ Fundamentals

Use this skill whenever the user asks about harmonic mixing, key matching, Camelot wheel, DJ transitions, mixing by key, energy flow in sets, or related DJ technique.

## Source Material

Primary reference: YouTube video `bw0MSJ3oVEQ` — "Harmonic Mixing for Beginners" style content covering Camelot wheel, key compatibility, and transition techniques.

## 1. The Camelot Wheel

The Camelot wheel maps musical keys to a circular, numbered notation that makes harmonic relationships obvious.

- Keys are labeled `X.Y` where `X` is the Camelot number (1–12) and `Y` is the flavor (A = major, B = minor).
- Moving clockwise adds energy; moving counter-clockwise subtracts energy.
- Same position is the strongest match (e.g., `8A` with `8A`).
- Adjacent positions (±1) are compatible with various effects.
- Relative major/minor pairs share the same number (e.g., `8A` is compatible with `8B`).

## 2. Key Compatibility Rules

| Move | Camelot Change | Compatibility | Effect |
|------|---------------|---------------|--------|
| Perfect match | 0 | `X.A` ↔ `X.A` / `X.B` ↔ `X.B` | Seamless |
| Relative major/minor | 0 | `X.A` ↔ `X.B` | Seamless |
| Energy up | +1 | `X.A` → `(X+1).A` | Uplifting |
| Energy down | −1 | `X.A` → `(X−1).A` | Deeper |
| Dominant jump | +1 flavor | `X.A` → `X.B` or `X.B` → `X.A` | Tension/release |
| Wheel skip (avoid) | ±3 | `X.A` → `X±3.A` | Dissonant / clash |

- Diametrically opposite keys on the wheel (e.g., `8A` ↔ `2A`) are generally incompatible.
- Mixed-key exceptions exist (e.g., `12A` is functionally compatible with `1A` due to the circular nature of pitch).

## 3. Mix Techniques by Category

### Filter Techniques
- **Low-pass filter sweep**: Cut highs during transition, then open; works across compatible keys.
- **High-pass filter sweep**: Opposite — remove lows for atmosphere, then drop bass back in.
- **Band-pass filter**: Isolate midrange for "vocal-only" moments before dropping the full beat.

### Time-Based Techniques
- **Echo / delay trails**: Let the outgoing track’s tail echo into the incoming track.
- **Reverb send**: Increase reverb on the outgoing track during the last 8–16 bars to wash it out smoothly.
- **Volume automation**: Ride the fader of the incoming track up while pulling the outgoing track down.

### Structural Techniques
- **Build-up → break → drop**: Use filtered or stripped-down versions of the next track during the build, then drop the full arrangement.
- **Loop + filter**: Loop a memorable phrase from the incoming track while filtering, then cut the loop and let the track play.
- **Track +1 / −1**: Use the previous or next track on the Camelot wheel as a “safe” transition when in doubt.

## 4. Energy Flow Management

Treat every DJ set as an energy arc:

1. **Opening**: warm, inviting, moderate BPM. Tend toward lower-energy keys.
2. **Ramp**: gradually increase BPM and move clockwise on the Camelot wheel.
3. **Peak**: highest energy, often the most compatible key cluster for the audience’s mood.
4. **Cool-down**: clockwise → counter-clockwise moves or same-key deeper tracks to ground the set before closing.

## 5. Practical Workflow

When the user asks for a mix plan, key suggestions, or transition ideas:

1. Identify their current track’s key (or Camelot notation if provided).
2. Suggest 2–3 next-track key zones using the compatibility table above.
3. Recommend a transition technique based on the genre and genre conventions.
4. Flag any risky key jumps (>1 step, opposite side of wheel) and propose alternatives.
5. Provide an energy narrative (rising / peak / cooling) if requested.

## 6. Pitfalls

- **Key detection errors**: Many track databases (Mixed In Key, Beatport, iTunes) disagree. If the user provides a key, trust their detection source and do not re-tag silently.
- **Genre conventions**: Some genres (e.g., techno, DnB) tolerate key drift better than others (e.g., pop, vocal house). Ask for the genre if it affects recommendations.
- **Energy ≠ Key**: A track one steps clockwise can still be lower energy; BPM and arrangement matter. Combine Camelot guidance with BPM and “vibe” checks.
