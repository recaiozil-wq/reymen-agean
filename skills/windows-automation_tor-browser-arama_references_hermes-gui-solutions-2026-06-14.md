# Hermes GUI Çözümleri — GitHub Araştırması

Tarih: 2026-06-14
Kaynak: Tor Browser ile GitHub araması

## En Kapsamlı Çözümler

### 1. hermes-studio (7.890★)
- https://github.com/EKKOLearnAI/hermes-studio
- Web dashboard + Desktop app (Windows/macOS/Linux)
- npm ile kurulum: `npm install -g hermes-web-ui && hermes-web-ui start`
- Vue3 + TypeScript
- Chat, session, profile, cron, file browser, terminal, analytics
- Platform channels: Telegram, Discord, Slack, WhatsApp, Matrix, Feishu, WeChat, WeCom

### 2. hermes-control-interface (762★)
- https://github.com/xaspx/hermes-control-interface
- Self-hosted web dashboard (localhost:10274)
- Vanilla JS + Vite + Node.js + WebSocket
- Chat, agents, MCP control, skills browser, token analytics, backup
- `git clone + npm install + node server.js`

### 3. Hermes-Studio (195★)
- https://github.com/JPeetz/Hermes-Studio
- React + TypeScript + PWA
- Multi-agent orchestration, memory, skills, approvals
- Self-hosted

### 4. hermesd (92★)
- https://github.com/mudrii/hermesd
- Python TUI (terminal içi dashboard)
- Hafif, bağımlılık yok

### 5. hermes-webui (106★)
- https://github.com/sanchomuzax/hermes-webui
- Process monitoring + configuration dashboard

## Seçim Kriterleri
- Windows'ta çalışmalı → hermes-studio (desktop app) veya hermes-control-interface (web)
- Kurulum kolaylığı → hermes-studio (tek npm komutu)
- Kapsam → hermes-studio (en geniş özellik seti)
- Hafiflik → hermes-control-interface (daha az bağımlılık)
