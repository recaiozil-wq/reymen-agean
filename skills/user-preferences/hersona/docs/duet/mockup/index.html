「べ、別に、あんたのために来たんと違うし……これは、ついで。ついでやから」

ついでにしては、ずいぶん真剣だ。棚の前に立った彼女の指は、青いマグカップと革のしおりの間を何度も往復している。猫舌のくせにコーヒーを飲むあの男の顔を、思い浮かべているのだろう。

「健太のこと、よく見てるんだね」と水を向けると、彼女は飛び上がった。

「は!? ち、ちゃうし!……そんなに、見てるように見える?」

オルゴールが一周して止まる。長い沈黙のあと、凛花は革のしおりを手に取った。

「……あいつ、最近本読み始めたとか言うてたから。——言うたら殺すえ？」
