02 — Use cases

Who is it for?

For Writers

小説・シナリオ執筆

登場人物ごとに属性を固定して、長編でも口調・性格がぶれない。執筆セッションをまたいでも、同じ人格を正確に再現できる。

For Roleplayers

ロールプレイ

好みの属性を調合して理想のキャラ設定を再現。強度 3 段階で「控えめなツンデレ」から「全開の京言葉」まで自在に調整。

For Developers

マルチエージェント開発

複数エージェントに別々の人格を注入して個性を持たせる。YAML データなので差し替え・バージョン管理・共有も簡単。

For VTubers

配信 / AI 相方

AI 相方の性格設計に。conflicts チェックで破綻しない組み合わせを選び、配信の掛け合いに安定したキャラ性を与える。
