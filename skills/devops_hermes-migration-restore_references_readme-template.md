# README Template — hermes-full-backup

Bu referans, README.md'nin güncel halini tutar. Repo içindeki README.md buna göre güncellenir.

## İçermesi Gereken Başlıklar

1. **Ana başlık**: `HERMES FULL BACKUP` + tek link
2. **İçindekiler tablosu**: Tüm dosya/klasörler + açıklamaları
3. **Yeni bilgisayara kurulum**: 5 adım (clone → restore → env → başlat)
4. **Güncelleme**: `-Update` parametreli kullanım
5. **Geri yüklenenler tablosu**: Kaynak → hedef eşlemesi
6. **Güvenlik uyarısı**: .env repoya dahil değil, push protection
7. **Notlar**: state.zip ~110MB, Hermes önceden kurulu olmalı
