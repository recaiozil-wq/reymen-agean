# Katkida Bulunma

ReYMeN acik kaynak bir projedir. Katkida bulunmak icin:

1. Fork et
2. pip install -e .[dev]
3. pre-commit install
4. Branch ac: feature/ozellik-adi
5. Kodla + test ekle
6. PR ac
