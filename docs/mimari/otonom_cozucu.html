⚙️ get_active_adapter() model
Hangi model aktifse onu alır — Ollama, LM Studio, GLM, DeepSeek, Claude, Gemini...
REYMEN_MODEL env var ile seçilir:
• auto → sırayla yerel servisleri dener
• ollama:llama3 → direkt Ollama
• deepseek:deepseek-coder → DeepSeek API
• lmstudio:local → LM Studio

Tüm adapter'lar aynı interface: complete(prompt) → str
🔄 Adım döngüsü run→check→fix
Her adımda script çalışır → exit code kontrolü → hata varsa düzeltme
1. run_script("reymen/scripts/step_N.py")
2. exit code = 0 ✅ → sonraki adım
3. exit code ≠ 0 ❌ → stderr mesajını al
4. ask_model_to_fix(code, error) ile hata mesajını LLM'e gönder
5. LLM'den gelen düzeltilmiş kodu fix/step_N_fix_vK.py'ye yaz
6. Tekrar çalıştır (max 3)
7. 3'te de başarısız → fail_log.jsonl → sonraki adım
