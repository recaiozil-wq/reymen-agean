# tor tarayıcıyı ac ekranda

Tor Browser sisteminizde kurulu görünmüyor. Tor Browser'ı kullanmak için:\n\n1. https://www.torproject.org/download/ adresinden indirin\n2. İndirilen dosyayı çalıştırıp kurulumu tamamlayın\n3. Kurulumdan sonra 'start-tor-browser' veya 'Tor Browser' komutuyla çalıştırabilirsiniz\n\nAlternatif olarak, otomatik kurulum için:\n- Windows: winget install TorProject.TorBrowser\n- Linux: sudo apt install torbrowser-launcher (veya flatpak/flathub)\n- macOS: brew install --cask tor-browser
