## Stack
- Frontend: Next.js 14+ (React, TypeScript, TailwindCSS)
- Backend: Next.js API routes (monorepo)
- Database: PostgreSQL (Railway)
- Cache/Queue: Redis (Railway)
- Storage: Cloudflare R2 (resim/CDN)
- Email: Resend
- Deployment: Railway (Dev + Production)
- Testing: Playwright
